Custom Thumbnails v1.1.0

Last update: Jan 22th, 2017
3 m

Description

Custom Thumbnails Module show HTMl content in a thumbnail format. You can configure the module with the following options:

  • Select number of content to shown.
  • Select frontend framework.
  • For each content you can:
    • Show or hide title of content and if you want it to have a link.
    • Show or hide content.
    • Select external or internal link.
    • Show or hide image and if you want it to have a link.
    • Show or hide button of "Read more" and the text of de button.

Versions

Version 1.0.0 stable

Version 1.0.0 of the module is compatible with the following Frontend frameworks:

  • Integration with Bootstrap 2.3.2 Framework. The same version on which Joomla 3.x version is based.
  • Integration with Bootstrap 3 Framework. The same version on which most of the Joomla Template Frameworks are based.
  • Integration with UiKit Framework.

Version 1.1.0 stable

  • Add Update Servers: Now you will be informed from the back of Joomla when there is a new update.
