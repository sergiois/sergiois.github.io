CF Gist

Versión estable: 1.0.2
Última actualización: 18 Enero, 2021

3 - cf

Campo personalizado para Joomla para integrar Gist de GitHub en artículos Joomla.

Instalación y uso

  • Instalar el plugin utilizando el gestor de extensiones de Joomla!
  • Habilitar el plugin
  • Añadir el campo (Gist) a tu componente
  • Introducir la URL de Gist:
    Ejemplo: https://gist.github.com/sergiois/afa88e675aca16f291065d7195f94651
